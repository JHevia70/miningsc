# MiningSC Overlay — Instalación

## Requisitos
- Windows 10/11 (64-bit)
- Star Citizen instalado y ejecutándose

## Instalación
1. Descomprime el ZIP en cualquier carpeta (p.ej. `C:\MiningSC\`)
2. Ejecuta `MiningOverlay.exe` — Windows pedirá permisos de administrador (necesarios para los atajos de teclado globales)
3. El icono de MiningSC aparecerá en la barra de tareas (system tray)

## Uso
| Tecla | Acción |
|-------|--------|
| F9    | Escanear pantalla y mostrar resultados |
| F10   | Mostrar / ocultar el overlay |
| Alt + arrastrar | Mover el overlay |

- Haz **clic derecho** en el icono del tray para acceder a la configuración o salir
- El overlay aparece en la esquina superior derecha de la pantalla
- Abre el panel de resultados de minería en Star Citizen antes de pulsar F9

## Configuración
Desde el menú de tray → **Configuración** puedes ajustar:
- Transparencia del overlay
- Tamaño de fuente
- Alineación del texto (izquierda / derecha)
- Directorio temporal para capturas
- Activar / desactivar el envío de datos a MiningSC

## Solución de problemas
- **F9 no responde**: asegúrate de que el .exe se ejecuta como administrador
- **Fuente extraña**: la fuente Electrolize se carga automáticamente, no necesitas instalarla
- **Overlay no visible**: pulsa F10 para mostrarlo, o usa el tray → "Mostrar / Ocultar"
